Chapter One Terms and Definitions

Code
Meaningful combinations of text and symbols that a web browser interprets to display the webpage.
Internet
A worldwide collection of computers linked together for use by businesses, governments, eduacational institutions, other organizations, and individuals using modems, phone lines, television cables, satellite links, fiber-optic connections, radio waves, and other communications devices and media.
Node
Any device, such as a computer, tablet, or smartphone, connected to a network.
Network
A collection of two or more computers linked together to share resources and information.
Internet of Things
The ever-growing number of devices connecting to a network, including appliances.
Data Lines
Lines that allow data to move from one computer to another.
Internet Backbone
A collection of high-speed data lines that link
Internet Service Provider (ISP)
A company that has a permanent connection to the Internet backbone.
World Wide Web
Service that provides access to information stored on web servers, the high-capacity, high-performance computers that power the web.
Webpages
Collection of linked files.
Website
A related collection of webpages created and maintained by a person, company, organization, or eduacational institution.
Home Page
Main page and first document people see on a website.
Hyperlink
More commonly called link, is an element that connects one webpage to another on the same server.
Protocol
A set of rules that defines how a client workstation can communicate with a server.
Server
The server is the host computere that stores resources and files for websites.
Hypertext Transfer Protcol (HHTP)
The main protocol used on the web to exchange and transfer webpages.
File Transfer Protocol (FTP)
Exchange files from one computer to another over the Internet.
Transmission Control Protocol/Internet Protcol (TCP/IP)
Overs the network connection between the data source and destination and micromanages the data.
Internet Protocol(IP)
Ensures data is sent to the correction location.
Domain Name Systems (DNS)
Created to resolve to resolve the issue of remembering and using IP addresses. Associates an IP address with a domain name.
Web Browser
Interprets and displays webpages so you can view and interact with them.
Domain
Area of the Internet a particular organization or person manages.
Internet Site
Website that is generally available to anyone with an Internet connection.
Extranet
Business information with select corporate partners or key customers.
Electronic Commerce (e-commerce)
Buying and selling goods and services on the Internet.
Learning Management System (LMS)
Web-based software application designed to facilitate online learning.
Responsive Design
Allows you to create one website that provides an optimal viewing experience.
Wirefram
Visual gude that idetifies the location of main webpage elements.
Active White Space
Area of the page that is intentionally left blank.
Passive White Space
Space between content areas.
Site Map
Planning tool that lists or displays all the pages on a website and indicates how they are related to each other.
Linear Website Structure
Allows webpages to be connected in a straight line.
Hierarchical Website Structure
Connects webpages in a treelike structure.
Webbed Website Structure
There is no structure and users are allowed too move between pages, even if the pages are located in different sections of a website.
World Wide Web Consortium (W3C)
An organization that develops and maintains standards, language specifications, and accessibiliyty recommendations.
Web Content
Information in a webpage that range from images, text, forms, and sounds.
Perceivable
Information and user interface components must be presentable to users in ways they can Perceive.
Operable
Users must be able to operate the interface.
Understandable
Information and the operation of the interface must be understandable.
Robust
Content must be robust enough to be interpreted by a variety of users.
Hypertext Markup Language (HTML)
Authoring languageused to create documents for the web.
Markup Language
HTML tags "mark up" or define the content on the webpage.
HTML Element
Consists of everything from the start tag to the end tag.
Attributes
Define additional characteristics of HTML elements such as height and width.
Syntax
Determines how elements should be written so the browser can interpret it correctly.
Paired Tags
HTML elements that use start tags and end tags.
Empty/Void Tags
HTML elements void of content.

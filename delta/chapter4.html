Cascading style sheets (CSS)A language you use to describe the formatting of a document written in a markup language such as HTML 5.
Style ruleA rule that defines the appearance of an element on a webpage.
Style sheetThe set of CSS style rules.
Inline styleYou add a style to the start tag for an element, such as a heading or paragraph.
Style attributeThe style changes the content marked up by a specific pair of tags, but does not affect other content in the document.
Embedded style sheetIncludes the style sheet within the opening and closing tags of the HTML document.
External style sheetA CSS file that contains all of the styles you want to apply to more than one page in the website.
ReskinningChanging the look of an entire website.
InheritanceCSS properties can be inherited from a parent element.
SpecificityDetermines which CSS rule to apply.
SelectorThe part of the statement that identifies what to style.
DeclarationDefines the exact formatting of the style.
PropertyIdentifies the style quality or characteristic to apply, such as color (text color), background-color, text-align, border-width, or font-size.
ValueIdentifies the particular property value to apply, such as green for color or 150% for font-size.
Fallback valuesThe additional values.
Font stackSpecifying two or more font values.
HexadecimalValues consist of a six-digit number code that corresponds to RGB (Red, Green, Blue) color values.
ChannelContains the number of red, green, or blue pixels necessary to create a specified color.
Block elementAppears as a block because it starts and ends with a new line, such as the main element or a paragraph element.
Inline elementsDisplayed without line breaks, so they flow within the same line.
CSS box modelDescribes these boxes of content on a webpage.
MarginProvides passive white space between block elements or between the top or bottom of a webpage.
BorderSeparates the padding and the margin of the block element.
PaddingThe passive white space between the content and the border of a block element.
Id selectorUses the id attribute value of an HTML element to select a single element.
Class selectorTo select elements that include a certain class attribute.
Descendant selectorList the elements in the order they appear to create the selector in the style rule.
Float propertyAllows you to position an element to the right or left of other elements, while remaining within its parent element.
Clear propertyTo remove the float effect.
CSS overflow propertyTo keep the content within its parent container.
List-style propertiesTo control the appearance of numbered and bulleted lists.
Responsive imageResizes itself to accommodate the size of the browser window.
Hero imageA large banner image strategically placed on the website to capture the visitorâ€™s attention.
Class attributeA class atrribute name can be applied to more than one div or other HTML elements on a webpage.
CSS resetA style rule that is applied to the webpage before any other style rule defined within the style sheet.

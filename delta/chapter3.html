GIFThe oldest file format and supports transparency and frame animation.
Lossless compressionMaintains the file's color information
PNGThe PNG file format was designed to replace the GIF file format for web graphics.
JPGThis is the standard file format for a digital photo, such as one taken with a digital camera.
Lossy compressionA lossy compression discards some of the color information in the image, which reduces its original quality.
SVGA format that uses markup language to create two-dimensional graphics, images, and animations.
PixelA term coined from “picture element,” is the smallest element of light or color on a device displaying images.
OptimizeOptimizing an image reduces its file size and load time.
Image tagAn empty HTML element used to add an image to a webpage.
SrcIdentifies the image file being inserted.
AltTo specify alternate text in case the image cannot be displayed in a browser
Div elementAn area or a division in a webpage.
IdIdentifies a unique area on a webpage and distinguishes it from other page divisions.
HyperlinksAllow a user to navigate a website and move from one page to another.
LinkText, an image, or other webpage content that visitors tap or click to instruct the browser to go to a location in a file or to request a file from a server.
Anchor elementYou use an anchor element to create a hyperlink on a webpage.
HrefIn the starting anchor tag to identify the webpage, email address, file, telephone number, or other content to access.
Relative linksHyperlinks that link to other webpages within the same website.
Absolute linksHyperlinks that link to other webpages outside of your website.
Email linksHyperlinks that link to an email address.
Telephone linksHyperlinks that link to a telephone number.
Unordered listDisplays a small graphic called a bullet before each item of information.
Ordered listDisplays information in a series using numbers or letters.
Description listContains terms and descriptions.
IframeTo embed a document within a webpage.

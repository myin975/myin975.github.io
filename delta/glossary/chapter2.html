Chapter Two Terms and Definitions

Template
Predefined webpage that contains a specific HTML sturcture to be used by all pages on a website.
Wireframe
The layout of a webpage, including its major content areas.
Semantic Wireframe
Wireframe that uses semantic elements to define the structure of a webpage.
Root Folder
The main folder for files. Contains CSS, images, media, and scripts.
Header Element
Element that structurally defines the header area.
Nav Element
Defines the navigation area of a webpage.
Main Element
Defines the main content area of a webpage.
Footer Element
Defines the bottom, or footer area of a webpage.
Word Wrap
Feature that causes text lines to break at the right edget of the window and appear on a new line, so all entered text is visible in the Notepad++ window.
Heading Elements
Elements that provide a title or heading before a paragraph of text or section of a page.
HTML Character Entity
Displays a reserved HTML character on the webpage.
Entity Name
An abbreviated name.
Entity Number
A combination of the pound sign and a numeric code.

Chapter Two Terms and Definitions

TemplatePredefined webpage that contains a specific HTML sturcture to be used by all pages on a website.
WireframeThe layout of a webpage, including its major content areas.
Semantic WireframeWireframe that uses semantic elements to define the structure of a webpage.
Root FolderThe main folder for files. Contains CSS, images, media, and scripts.
Header ElementElement that structurally defines the header area.
Nav ElementDefines the navigation area of a webpage.
Main Element
Defines the main content area of a webpage.
Footer ElementDefines the bottom, or footer area of a webpage.
Word WrapFeature that causes text lines to break at the right edget of the window and appear on a new line, so all entered text is visible in the Notepad++ window.
Heading ElementsElements that provide a title or heading before a paragraph of text or section of a page.
HTML Character EntityDisplays a reserved HTML character on the webpage.
Entity NameAn abbreviated name.
Entity NumberA combination of the pound sign and a numeric code.
